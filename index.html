About Meridian

Meridian is a fully client-side investment analysis platform. All data is fetched live from Yahoo Finance. No backend server, no API key, no login required.

Portfolio and watchlist data is stored in your browser's localStorage. Clearing your browser data will erase it.

Indicators Used
RSI MACD Bollinger EMA 200/100/50/25 Ichimoku ATR Ann. Volatility
Data Management
Removes portfolio, watchlist and settings from localStorage.
